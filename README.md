A simple python program which get our wifi essid from linux terminal.

### Setup ###

+ sudo pip install -r requirements.txt

### Terminal Command ###

+ git clone https://github.com/r3k4t/getwifiessid.git
+ cd getwifiessid
+ sudo python getwifiessid.py
